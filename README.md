# 🏆 Gamified Academic Platform (Arena)

A full-stack MERN application designed to gamify the academic experience. This platform replaces messy submission threads and scattered links by providing a dedicated arena where professors can deploy assignments as "Quests" and students can earn XP to climb the leaderboard.

## 🚀 Live Demo
- **Live Platform:** [https://gamified-platform-delta.vercel.app](https://gamified-platform-delta.vercel.app)
- **Video Walkthrough:** [Watch Demo on LinkedIn](https://www.linkedin.com/in/harshit-bisht-dev/)

## ✨ Key Features

### 👨‍🏫 Professor OS (Admin)
- **Classroom Management:** Create classrooms and generate unique 6-digit join codes.
- **Quest Deployment:** Create and deploy coding challenges or assignments with specific XP rewards.
- **Grading Desk:** One-click review system to evaluate student submissions (code links/images), approve them, and award XP.

### 🎓 Student OS (User)
- **Enrollment:** Join classrooms using the professor's unique code.
- **Submissions:** Upload assignment proofs via Cloudinary (images/PDFs) or submit GitHub repository links.
- **Dynamic Leaderboard:** Track XP in real-time and compete globally against other students.

## 💻 Tech Stack

- **Frontend:** React.js (Vite), Chakra UI
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** JSON Web Tokens (JWT)
- **File Storage:** Multer, Cloudinary
